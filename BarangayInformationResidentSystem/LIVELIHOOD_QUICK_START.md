# Livelihood & Skills Registry - Quick Start Guide

## Getting Started in 5 Minutes

### Step 1: Access the Feature
1. Login to your BIRS account
2. Look for the **"Economic Development"** section in the left sidebar
3. You'll see 4 new options:
   - Livelihood & Skills
   - Training Sessions
   - Job Opportunities
   - Livelihood Reports

### Step 2: Add Your First Resident Skill

**Goal:** Document that Juan Dela Cruz knows carpentry

1. Click **"Livelihood & Skills"**
2. You're on the **"Resident Skills"** tab by default
3. Click **"Add Skill to Resident"** button (blue button, top right)
4. A popup form appears:
   - **Resident:** Select "Juan Dela Cruz"
   - **Skill:** Select "Carpentry"
   - **Proficiency Level:** Choose "Advanced"
   - **Years of Experience:** Enter "8"
   - **Certification:** Type "TESDA Certificate 2020"
   - **Notes:** Add any additional info (optional)
5. Click **"Add Skill"**
6. Success! You'll see a green message at the top

### Step 3: Create Your First Livelihood Program

**Goal:** Create a carpentry training program

1. On the same page, click the **"Livelihood Programs"** tab
2. Click **"Add Program"** button
3. Fill in the form:
   - **Program Name:** "Carpentry Skills Development 2025"
   - **Description:** "A comprehensive carpentry training program for beginners and intermediate learners"
   - **Program Type:** Select "Training"
   - **Status:** Select "Planning"
   - **Start Date:** Pick a date (e.g., 2025-02-01)
   - **End Date:** Pick an end date (optional)
   - **Location:** "Barangay Hall"
   - **Target Beneficiaries:** "20"
   - **Budget:** "50000"
4. Click **"Add Program"**
5. Your program appears as a card on the page

### Step 4: Create a Training Session

**Goal:** Schedule the first training session

1. Click **"Training Sessions"** in the sidebar
2. Click **"Add Training Session"** button
3. Fill in the form:
   - **Program:** Select "Carpentry Skills Development 2025"
   - **Session Name:** "Basic Carpentry Techniques"
   - **Related Skill:** Select "Carpentry"
   - **Trainer Name:** "Mr. Jose Santos"
   - **Session Date:** Pick a date
   - **Session Time:** Pick a time (e.g., 09:00)
   - **Duration:** "8" hours
   - **Location:** "Barangay Hall - Room 1"
   - **Max Participants:** "20"
4. Click **"Add Session"**
5. Your session appears as a card

### Step 5: Enroll Participants

**Goal:** Add residents to the training session

1. On the session card, click **"Enroll Participant"**
2. A popup appears:
   - **Resident:** Select a resident (e.g., "Maria Santos")
3. Click **"Enroll"**
4. Repeat for more residents
5. Click **"View Participants"** to see the list

### Step 6: Track Training Progress

**Goal:** Mark attendance and completion

1. Click **"View Participants"** on a session card
2. A table shows all enrolled residents
3. For each resident:
   - **Attendance:** Change from "Present" to "Absent" or "Excused"
   - **Completion:** Change from "Not Started" to "In Progress" or "Completed"
   - **Certificate:** Mark as "Yes" when completed
4. Changes save automatically

### Step 7: Post a Job Opportunity

**Goal:** Post a job for skilled residents

1. Click **"Job Opportunities"** in the sidebar
2. Click **"Post Job Opportunity"** button
3. Fill in the form:
   - **Job Title:** "Carpenter - ABC Construction"
   - **Description:** "We need experienced carpenters for a residential project"
   - **Required Skills:** Type each skill on a new line:
     ```
     Carpentry
     Welding
     Safety Training
     ```
   - **Employer Name:** "ABC Construction Company"
   - **Employer Contact:** "555-1234"
   - **Location:** "Makati City"
   - **Salary Range:** "18,000 - 22,000"
   - **Employment Type:** "Full-time"
   - **Application Deadline:** Pick a date
4. Click **"Post Opportunity"**
5. The job appears on the page

### Step 8: Review Applications

**Goal:** See who applied for the job

1. Click the **"Applications"** tab
2. You'll see a table of all applications
3. For each application:
   - Click **"Update"** to change status
   - Options: Applied, Under Review, Shortlisted, Rejected, Hired, Withdrawn
   - Add notes about the candidate
4. Click **"Update Status"** to save

### Step 9: Generate Your First Report

**Goal:** See statistics about skills and programs

1. Click **"Livelihood Reports"** in the sidebar
2. You'll see report options at the top:
   - Skills Summary
   - Programs
   - Training
   - Jobs
   - Residents
   - All Reports
3. Click **"Skills Summary"**
4. You'll see:
   - All skills in the system
   - How many residents have each skill
   - Proficiency distribution (Beginner, Intermediate, Advanced, Expert)
5. Try other reports to see different data

## Common Tasks

### Add a Skill to Multiple Residents
1. Go to Livelihood & Skills
2. Click "Add Skill to Resident"
3. Select first resident and skill
4. Click "Add Skill"
5. Repeat for each resident

### Update a Resident's Skill Level
1. Go to Livelihood & Skills
2. Find the skill in the table
3. Click the delete button
4. Add the skill again with the new proficiency level

### Track Training Attendance
1. Go to Training Sessions
2. Click "View Participants" on a session
3. Change attendance status for each person
4. Changes save automatically

### Find Residents for a Job
1. Create a job opportunity with required skills
2. Go to Livelihood Reports ‚Üí Residents
3. Look for residents with matching skills
4. Contact them about the opportunity

## Troubleshooting

### "Resident not showing in dropdown"
- Make sure the resident is marked as "Active" in the Residents section
- Check if the resident exists in the system

### "Can't delete a program"
- Make sure you have admin permissions
- Check if the program has active sessions

### "Participant not appearing in list"
- Refresh the page
- Make sure the enrollment was successful
- Check the session ID is correct

### "Report showing no data"
- Make sure you've added skills/programs/training first
- Check that records are not archived
- Try a different report type
